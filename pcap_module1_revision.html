Παγίδες & Προσοχή

Οι λεπτομέρειες που τρώνε πόντους από οριακούς μαθητές. Αυτές είναι που χωρίζουν το «το ξέρω περίπου» από το «το πετυχαίνω σίγουρα».

📌 floor vs ceil vs trunc στους αρνητικούς

Σε θετικούς, floor και trunc δίνουν το ίδιο. Σε αρνητικούς, διαφέρουν: το trunc πάει προς το 0, το floor προς το −∞.

−2.5 → floor = −3, ceil = −2, trunc = −2

📌 randrange αποκλείει το πάνω όριο — το randint όχι

randrange(1, 10) δίνει 1…9 (το 10 ΠΟΤΕ). Αντίθετα, randint(1, 10) δίνει 1…10 (το 10 ΝΑΙ).

Μνημονικό: randint(a,b) = randrange(a, b+1)

📌 Ίδιος seed → ίδια αποτελέσματα

Ο γεννήτορας είναι ψευδοτυχαίος. Αν ξανα-ορίσεις τον ίδιο σπόρο, οι επόμενες κλήσεις δίνουν τα ίδια νούμερα — οπότε η διαφορά τους είναι 0.

import random
random.seed(0); x = random.choice([1, 2])
random.seed(0); y = random.choice([1, 2])
print(x - y)   # πάντα 0

📌 qualified vs unqualified όνομα

Το ποια μορφή κλήσης ισχύει εξαρτάται από το πώς έκανες import:

• Μετά από import a.b → υποχρεωτικά a.b.c() (όχι b.c(), όχι c()).
• Μετά από from a.b import c → σκέτο c().

Παγίδα: μετά τη λέξη import σε ένα from … import μπαίνουν ονόματα οντοτήτων, όχι module.function.

📌 Δεν «εισάγεις πακέτο» για να καλέσεις συνάρτηση κατευθείαν

Το import p από μόνο του δεν σου δίνει πρόσβαση στο p.m.f(). Πρέπει να φτάσεις μέχρι το module: import p.m ή from p.m import f.

📌 __name__ δεν είναι πάντα '__main__'

Είναι '__main__' μόνο όταν το αρχείο τρέχει ως κύριο πρόγραμμα. Όταν γίνεται import ως module, παίρνει το όνομα του module.

Γι' αυτό γράφουμε if __name__ == "__main__": για κώδικα που τρέχει μόνο standalone.

📌 __pycache__ & .pyc

Ο φάκελος __pycache__ δημιουργείται αυτόματα από την Python — ποτέ με το χέρι. Ο semi-compiled κώδικας αποθηκεύεται σε αρχεία .pyc (όχι .pby, όχι .pyo), με όνομα που περιέχει την έκδοση: mod.cpython-XY.pyc.

📌 Multi-select: ΜΗΔΕΝ partial credit

Στις ερωτήσεις «Select two», αν διαλέξεις 1 σωστή + 1 λάθος, ή 2 σωστές + 1 παραπάνω, παίρνεις 0 πόντους — όχι μισούς. Διάλεξε ακριβώς όσες ζητάει, ούτε μία παραπάνω/λιγότερη.

Τακτική: πρώτα αποκλείεις τις σίγουρα λάθος, μετά επιλέγεις ανάμεσα στις υπόλοιπες.
