What is true about Python packages?

  • A package is a group of related modules.
  • A package is a single file whose name ends with the extension .py.
  • The __name__ variable always contains the name of a package.
  • The pyc extension is used to mark semi-compiled Python packages.
Έννοια

Ένα package είναι ένας φάκελος που ομαδοποιεί σχετιζόμενα modules (αρχεία .py). Το .pyc είναι το αρχείο bytecode που παράγεται αυτόματα κατά το import — η ορολογία της PCAP το λέει «semi-compiled».

Ανάλυση

Με την εκτέλεση είδαμε ότι μετά από import δημιουργούνται αρχεία .pyc μέσα στο __pycache__, και ότι ένα package είναι φάκελος που περιέχει modules.

Γιατί όχι τα άλλα

Το __name__ περιέχει όνομα module και γίνεται '__main__' όταν τρέχει κατευθείαν. Ένα μεμονωμένο .py είναι module, όχι package.

📌
Παγίδα & Προσοχή. Μην μπερδεύεις module (αρχείο) με package (φάκελος + __init__.py).
